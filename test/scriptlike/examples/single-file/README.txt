To run this example:

First, ensure you have a DUB installed, and a D compiler such as DMD:
- DMD: http://dlang.org/download.html#dmd
- DUB: http://code.dlang.org/download

DUB must be at least v1.0.0.

And then...

On Linux/OSX:
-------------
$ chmod +x myscript.d
$ ./myscript.d
or
$ ./myscript.d Frank

On *any* OS:
-------------
$ dub myscript.d
or
$ dub myscript.d Frank
